- Single Ionic Angular app at repo root (web + Android + iOS via Capacitor).

- Procedural **scene** generation architecture aligned to authored story beats.

### Changed
- Consolidated documentation into domain folders under `/docs/`.
- Updated Golden Record to the flooded-world JRPG scope (3-hour authored story).
